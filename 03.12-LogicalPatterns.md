## Pattern 41: Multiplication Table

### Example

```text

5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
```

Reverse the digits of a number.

### Example

```text
12345 → 54321
```

### Logic

* Extract the last digit.
* Add it to the reversed number.
* Remove the last digit.
* Repeat until the number becomes 0.

```python
num = 12345
reverse = 0

    digit = num % 10
```

## Pattern 46: Palindrome Number

### Problem

Check whether a number is a palindrome.

### Example

```text
121 → Palindrome

123 → Not Palindrome
```

### Logic

* Reverse the number.
* Compare with the original number.

If both are equal:

```text
Palindrome
```

Otherwise:

```text
Not Palindrome
```

### Solution

```python
num = 121
original = num
reverse = 0

while num > 0:

    digit = num % 10

    reverse = reverse * 10 + digit

    num = num // 10

if original == reverse:
    print("Palindrome")
else:
    print("Not Palindrome")
```

## Pattern 48: Guessing Game

### Problem

Keep asking the user to guess a number.

### Example

```text
Enter number:

Too High

Too Low

Correct
```

### Logic

* Store a secret number.
* Compare the user's guess.
* Continue until the correct number is entered.

### Solution

```python
secret = 7

while True:

    guess = int(input("Enter number: "))

    if guess > secret:
        print("Too High")

    elif guess < secret:
        print("Too Low")

    else:
        print("Correct")
        break
```

## Pattern 51: Armstrong Number

### Problem

Check whether a number is an Armstrong Number.

### Definition

A number is an Armstrong Number if:

```text
Sum of (digits ^ number_of_digits)
=
Original Number
```

### Example

```text
153

1³ + 5³ + 3³

= 1 + 125 + 27

= 153
```

Therefore:

```text
153 → Armstrong Number
```

### Logic

1. Count digits.
2. Extract each digit.
3. Raise the digit to the power of digit count.
4. Add the results.
5. Compare with the original number.

### Solution

```python
num = 153
original = num

digits = len(str(num))

total = 0

while num > 0:

    digit = num % 10

    total += digit ** digits

    num = num // 10

if total == original:
    print("Armstrong Number")
else:
    print("Not Armstrong Number")
```

## Pattern 53: Roman to Integer

### Problem

Convert a Roman numeral into an integer.

### Examples

```text
I  = 1

IV = 4

IX = 9

X  = 10
```

### Roman Symbols

| Symbol | Value |
| ------ | ----- |
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |

---

### Logic

Normally:

```text
VI

5 + 1 = 6
```

Special cases:

```text
IV

5 - 1 = 4
```

```text
IX

10 - 1 = 9
```

If the current symbol is smaller than the next symbol:

```text
Subtract
```

Otherwise:

```text
Add
```

### Solution

```python
roman = "IX"

values = {
    'I': 1,
    'V': 5,
    'X': 10,
    'L': 50,
    'C': 100,
    'D': 500,
    'M': 1000
}

total = 0

for i in range(len(roman)):

    if (
        i < len(roman) - 1
        and values[roman[i]] < values[roman[i + 1]]
    ):
        total -= values[roman[i]]
    else:
        total += values[roman[i]]

print(total)
```

Output:

```text
9
```

## Pattern 56: Valid Parentheses

### Problem

Check whether parentheses are balanced.

### Examples

```text
()     → Valid

([])   → Valid

(]     → Invalid
```

### Why Use a Stack?

A Stack follows:

```text
Last In First Out (LIFO)
```

Whenever we see:

```text
(
[
{
```

Push into the stack.

Whenever we see:

```text
)
]
}
```

Pop from the stack and check if it matches.

---

### Logic

```text
Opening Bracket
→ Push

Closing Bracket
→ Check Top of Stack

Match?
→ Continue

No Match?
→ Invalid
```

---

### Solution

```python
s = "([])"

stack = []

pairs = {
    ')': '(',
    ']': '[',
    '}': '{'
}

valid = True

for ch in s:

    if ch in "([{":
        stack.append(ch)

    else:

        if (
            not stack
            or stack.pop() != pairs[ch]
        ):
            valid = False
            break

if valid and not stack:
    print("Valid")
else:
    print("Invalid")
```

### Dry Run

Input:

```text
([])
```

Process:

```text
( → Push

[ → Push

] → Match [

) → Match (
```

Stack becomes empty.

Output:

```text
Valid
```
