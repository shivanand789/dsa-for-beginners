# Hollow Patterns

In all previous patterns, we printed every character inside the shape.

For Hollow Patterns, we only print the **boundary** and leave the inside empty.

---

## Boundary Rule

Print the character only when the current position lies on the boundary.

A cell is considered a boundary if it belongs to:

* Top Boundary
* Bottom Boundary
* Left Boundary
* Right Boundary

Otherwise, print a space.

---

### Visual Representation

```text
*****
*   *
*   *
*   *
*****
```

Boundary cells:

```text
Top Row
Bottom Row
First Column
Last Column
```

Inner cells:

```text
Print Space
```

---

## Universal Boundary Condition

For most hollow patterns:

```python
if (
    first_row
    or last_row
    or first_column
    or last_column
):
    print("*")
else:
    print(" ")
```

---

## Pattern 32: Hollow Square

### Output

```text
*****
*   *
*   *
*   *
*****
```

---

### Observation

Rows = 5

Columns = 5

Print stars only on:

* First row
* Last row
* First column
* Last column

---

### Boundary Conditions

```text
Top Boundary    → row == 0

Bottom Boundary → row == rows - 1

Left Boundary   → col == 0

Right Boundary  → col == cols - 1
```

---

### For Loop

```python
rows = 5
cols = 5

for row in range(rows):

    for col in range(cols):

        if (
            row == 0
            or row == rows - 1
            or col == 0
            or col == cols - 1
        ):
            print("*", end="")
        else:
            print(" ", end="")

    print()
```

---

### While Loop

```python
rows = 5
cols = 5

row = 0

while row < rows:

    col = 0

    while col < cols:

        if (
            row == 0
            or row == rows - 1
            or col == 0
            or col == cols - 1
        ):
            print("*", end="")
        else:
            print(" ", end="")

        col += 1

    print()
    row += 1
```

## Pattern 33: Hollow Triangle

### Output

```text
    *
   * *
  *   *
 *     *
*********
```

---

### Row Analysis

| Row | Output    |
| --- | --------- |
| 1   | *         |
| 2   | * *       |
| 3   | *   *     |
| 4   | *     *   |
| 5   | ********* |

Notice:

* First and last stars form the boundary.
* Last row is completely filled.

---

### Boundary Conditions

A star is printed when:

```text
First Star of Row

OR

Last Star of Row

OR

Bottom Row
```

---

### For Loop

```python
rows = 5

for row in range(rows):

    # Spaces
    for space in range(rows - row - 1):
        print(" ", end="")

    # Pyramid Width
    for col in range(2 * row + 1):

        if (
            col == 0
            or col == 2 * row
            or row == rows - 1
        ):
            print("*", end="")
        else:
            print(" ", end="")

    print()
```

---

### While Loop

```python
rows = 5

row = 0

while row < rows:

    spaces = rows - row - 1

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    col = 0

    while col < (2 * row + 1):

        if (
            col == 0
            or col == 2 * row
            or row == rows - 1
        ):
            print("*", end="")
        else:
            print(" ", end="")

        col += 1

    print()
    row += 1
```
