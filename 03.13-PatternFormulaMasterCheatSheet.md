## Hollow Pattern Formulas

### Universal Boundary Rule

Print the symbol only if the current position lies on a boundary.

```python
if boundary:
    print("*")
else:
    print(" ")
```

### Hollow Square

```python
row == 0
or row == rows - 1
or col == 0
or col == cols - 1
```

### Hollow Pyramid

```python
col == 0
or col == width - 1
or row == rows - 1
```

### Hollow Pattern Formula

```text
Boundary + Spaces
```
