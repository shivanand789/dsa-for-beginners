## Pattern 28: Inverted Star Pyramid

### Output

```text
*********
 *******
  *****
   ***
    *
```

### Observation

| Row | Spaces | Stars |
| --- | ------ | ----- |
| 1   | 0      | 9     |
| 2   | 1      | 7     |
| 3   | 2      | 5     |
| 4   | 3      | 3     |
| 5   | 4      | 1     |

Notice:

* Spaces increase by 1.
* Stars decrease by 2.

### Formula

```text
Spaces = row - 1

Stars = 2 × (rows - row) + 1
```
---

### For Loop

```python
rows = 5

for row in range(1, rows + 1):

    # Spaces
    for space in range(row - 1):
        print(" ", end="")

    # Stars
    for star in range(2 * (rows - row) + 1):
        print("*", end="")

    print()
```

---

### While Loop

```python
rows = 5
row = 1

while row <= rows:

    spaces = row - 1

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    stars = 2 * (rows - row) + 1

    while stars > 0:
        print("*", end="")
        stars -= 1

    print()
    row += 1
```

---

### Comparing Normal and Inverted Pyramid

#### Normal Pyramid

```text
    *
   ***
  *****
 *******
*********
```

Formula:

```text
Spaces = rows - row

Stars = 2 × row - 1
```

Notice:

* Spaces decrease
* Stars increase

---

#### Inverted Pyramid

```text
*********
 *******
  *****
   ***
    *
```

Formula:

```text
Spaces = row - 1

Stars = 2 × (rows - row) + 1
```

Notice:

* Spaces increase
* Stars decrease

## Pattern 29: Inverted Number Pyramid

### Output

```text
123454321
 1234321
  12321
   121
    1
```

### Observation

Look at the first row:

```text
123454321
```

The middle number is:

```text
5
```

Now look at the next row:

```text
1234321
```

The middle number becomes:

```text
4
```

---

### Pattern Analysis

| Row | Spaces | Middle Number | Output    |
| --- | ------ | ----------- | --------- |
| 1   | 0      | 5           | 123454321 |
| 2   | 1      | 4           | 1234321   |
| 3   | 2      | 3           | 12321     |
| 4   | 3      | 2           | 121       |
| 5   | 4      | 1           | 1         |

Notice:

* Spaces increase every row.
* Numbers increase from 1 to the Middle.
* Numbers then decrease back to 1.
* Middle number decreases every row.

---

### Logic

Each row has three parts:

#### Part 1: Spaces

```text
0, 1, 2, 3, 4
```

#### Part 2: Increasing Numbers

```text
1 → Peak
```

#### Part 3: Decreasing Numbers

```text
Peak - 1 → 1
```

---

### For Loop

```python
rows = 5

for row in range(rows, 0, -1):

    # Spaces
    for space in range(rows - row):
        print(" ", end="")

    # Increasing numbers
    for num in range(1, row + 1):
        print(num, end="")

    # Decreasing numbers
    for num in range(row - 1, 0, -1):
        print(num, end="")

    print()
```

---

### While Loop

```python
rows = 5
row = rows

while row >= 1:

    spaces = rows - row

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    num = 1

    while num <= row:
        print(num, end="")
        num += 1

    num = row - 1

    while num >= 1:
        print(num, end="")
        num -= 1

    print()
    row -= 1
```

---

### Normal vs Inverted Number Pyramid

#### Normal Number Pyramid

```python
for row in range(1, rows + 1):
```

Produces:

```text
    1
   121
  12321
 1234321
123454321
```

Peak number increases.

---

#### Inverted Number Pyramid

```python
for row in range(rows, 0, -1):
```

Produces:

```text
123454321
 1234321
  12321
   121
    1
```

Peak number decreases.

---

## Key Takeaways

### Inverted Star Pyramid

```text
Spaces ↑
Stars ↓
```

### Inverted Number Pyramid

```text
Spaces ↑
Peak Number ↓
```

### Quick Formula

```text
Normal Pyramid
--------------
Spaces = rows - row

Inverted Pyramid
----------------
Spaces = row - 1
```

A useful pattern rule to remember:

```text
Normal Pattern
→ Row increases

Inverted Pattern
→ Row decreases
```

Once you identify whether a pattern is normal or inverted, the solution becomes much easier.
