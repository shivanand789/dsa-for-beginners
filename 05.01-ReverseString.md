# Leet Code 344 -  Reverse a String

## Problem

Given a string `s`, return the string in reverse order.

### Example

```python
s = "hello"
```

**Output**

```python
"olleh"
```

---

## Idea: Two Pointers + Swap

Since strings are immutable in Python, convert the string into a list of characters.

Consider:

```text
[1, 2, 3, 4, 5]
 L           R
```

Swap the elements at the left and right pointers:

```text
1 ↔ 5

[5, 2, 3, 4, 1]
```

Move both pointers inward:

```text
[5, 2, 3, 4, 1]
    L     R
```

Continue swapping until:

```python
left >= right
```

---

## Code

```python
class Solution:
    def reverseString(self, s: List[str]) -> None:
        l = 0
        r = len(s) - 1

        while l < r:
            s[l], s[r] = s[r], s[l]

            l += 1
            r -= 1

        print(s)
```

---

## Dry Run

### Input

```python
s = ["h", "e", "l", "l", "o"]
```

Initial State:

```text
[h, e, l, l, o]
 L           R
```

### Iteration 1

```text
Swap h and o

[o, e, l, l, h]
    L     R
```

### Iteration 2

```text
Swap e and l

[o, l, l, e, h]
       LR
```

Stop because:

```python
left >= right
```

### Result

```python
["o", "l", "l", "e", "h"]
```

---

## Complexity Analysis

### Time Complexity

```text
O(n)
```

Each character is visited at most once.

### Space Complexity

```text
O(1)
```

The reversal is performed in-place using only two pointers.

---

## Pattern Recognition

This is a classic **Two Pointer** problem.

```text
Left Pointer  → starts at beginning
Right Pointer → starts at end

Swap and move inward
```

This pattern is commonly used in:

* Reverse String
* Valid Palindrome
* Reverse Words
* Container With Most Water
* Sorted Array Problems
