# Level 5: Number Triangle Variations

In the previous levels, we learned:

* Star patterns

In this level, we will learn patterns where the numbers follow different sequences and rules.

The most important skill here is to identify:

# Pattern 20: Continuous Increasing Numbers

## Output

```text
1
23
456
78910
```

## Observation

| Row | Numbers Printed |
| --- | --------------- |
| 1   | 1               |
| 2   | 2 3             |
| 3   | 4 5 6           |
| 4   | 7 8 9 10        |

Notice:

* Row 1 contains 1 number.
* Row 2 contains 2 numbers.
* Row 3 contains 3 numbers.
* Row 4 contains 4 numbers.
* Numbers continue increasing across rows.

Unlike previous patterns, the numbering does not restart.

---

## Logic

We need a variable that keeps increasing.

```python
num = 1
```

Every time a number is printed:

```python
num += 1
```

---

## For Loop

```python
num = 1

for row in range(1, 5):

    for col in range(row):
        print(num, end=" ")
        num += 1

    print()
```

---

## While Loop

```python
row = 1
num = 1

while row <= 4:

    col = 1

    while col <= row:
        print(num, end=" ")
        num += 1
        col += 1

    print()
    row += 1
```

---

## Questions to Ask

1. How many rows?

   * 4

2. How many items in each row?

   * Equal to the row number

3. Is there a number that keeps increasing?

   * Yes

That increasing variable is the key to solving this pattern.

---

## General Template

```python
num = 1

for row in range(1, rows + 1):

    for col in range(row):
        print(num, end=" ")
        num += 1

    print()
```

# Pattern 21: Reverse Counting Triangle

## Output

```text
1
21
321
4321
54321
```

## Observation

| Row | Output |
| --- | ------ |
| 1   | 1      |
| 2   | 21     |
| 3   | 321    |
| 4   | 4321   |
| 5   | 54321  |

Notice:

* Each row starts with the row number.
* Numbers decrease until 1.

---

## Rule

* Start from the row number.
* Count backwards.
* Stop at 1.

---

## For Loop

```python
for row in range(1, 6):

    for col in range(row, 0, -1):
        print(col, end="")

    print()
```

---

## While Loop

```python
row = 1

while row <= 5:

    col = row

    while col >= 1:
        print(col, end="")
        col -= 1

    print()
    row += 1
```

---

## Questions to Ask

* What is the first number in each row?

  * Same as the row number.

* Does it increase or decrease?

  * Decrease.

* Where does it stop?

  * At 1.

# Pattern 22: Fixed Start Descending Numbers

## Output

```text
5
54
543
5432
54321
```

## Observation

| Row | Output |
| --- | ------ |
| 1   | 5      |
| 2   | 54     |
| 3   | 543    |
| 4   | 5432   |
| 5   | 54321  |

Notice:

* Every row starts with 5.
* Each row prints one additional number.
* Numbers decrease by 1.

---

## Logic

Ask:

### How many items should each row print?

```text
Row 1 → 1 item
Row 2 → 2 items
Row 3 → 3 items
Row 4 → 4 items
Row 5 → 5 items
```

### What is the first number?

```text
Always 5
```

### Should numbers increase or decrease?

```text
Decrease
```

Compare:

```text
12345 → Starts from 1, increases

54321 → Starts from 5, decreases
```

---

## For Loop

```python
for row in range(1, 6):

    for col in range(5, 5 - row, -1):
        print(col, end="")

    print()
```

---

## Easier Version

```python
for row in range(1, 6):

    count = 0
    num = 5

    while count < row:
        print(num, end="")
        num -= 1
        count += 1

    print()
```

---

## While Loop

```python
row = 1

while row <= 5:

    num = 5
    count = 1

    while count <= row:
        print(num, end="")
        num -= 1
        count += 1

    print()
    row += 1
```

# Pattern 23: Repeated Row Numbers

## Output

```text
1
22
333
4444
55555
```

## Observation

* Print the row number.
* Print it row-number times.

---

## Rule

```text
Row 1 → Print 1 once

Row 2 → Print 2 twice

Row 3 → Print 3 three times

Row 4 → Print 4 four times

Row 5 → Print 5 five times
```

---

## For Loop

```python
for row in range(1, 6):

    for col in range(row):
        print(row, end="")

    print()
```

---

## While Loop

```python
row = 1

while row <= 5:

    col = 1

    while col <= row:
        print(row, end="")
        col += 1

    print()
    row += 1
```

# Pattern 24: Repeated Alphabet Triangle

## Output

```text
A
BB
CCC
DDDD
EEEEE
```

## Observation

This pattern follows the same logic as Pattern 24.

The only difference is that we print alphabets instead of numbers.

---

## Rule

```text
Row 1 → Print A once

Row 2 → Print B twice

Row 3 → Print C three times

Row 4 → Print D four times

Row 5 → Print E five times
```

---

## For Loop

```python
for row in range(1, 6):

    ch = chr(64 + row)

    for col in range(row):
        print(ch, end="")

    print()
```

---

## While Loop

```python
row = 1

while row <= 5:

    ch = chr(64 + row)
    col = 1

    while col <= row:
        print(ch, end="")
        col += 1

    print()
    row += 1
```

---

## Easier Version

```python
letters = "ABCDE"

for row in range(1, 6):

    for col in range(row):
        print(letters[row - 1], end="")

    print()
```

# Pattern Analysis Summary

| Pattern                        | Starting Value | Direction  | Depends On     |
| ------------------------------ | -------------- | ---------- | -------------- |
| Continuous Numbers             | 1              | Increasing | Global Counter |
| Reverse Counting Triangle      | Row Number     | Decreasing | Row            |
| Fixed Start Descending Numbers | 5              | Decreasing | Row            |
| Repeated Numbers               | Row Number     | Constant   | Row            |
| Repeated Alphabets             | Row Letter     | Constant   | Row            |

---

# Key Takeaways

Before solving any number pattern, ask:

### Step 1

How many rows are there?

### Step 2

How many values should each row print?

### Step 3

What is the starting value?

### Step 4

Should the values:

* Increase?
* Decrease?
* Stay constant?

### Step 5

Does the pattern depend on:

* Row number?
* Column number?
* A separate counter variable?

Remember:

```text
Pattern = Rows + Columns + Logic
```

If you identify these three things correctly, most number patterns become easy to solve.
