# Level 4: Reverse Triangle Patterns

In the previous level, we learned how to create **increasing triangle patterns**.

In this level, we will learn how to create **reverse triangle patterns**.

The logic is almost the same.

The only difference is:

* Increasing Triangle → Rows increase from 1 to N
* Reverse Triangle → Rows decrease from N to 1

---

# Pattern 16: Reverse Star Triangle

## Output

```text
*****
****
***
**
*
```

## Observation

| Row | Stars |
| --- | ----- |
| 5   | ***** |
| 4   | ****  |
| 3   | ***   |
| 2   | **    |
| 1   | *     |

The number of stars decreases as the row number decreases.

---

## Logic

* Row 5 → Print 5 stars
* Row 4 → Print 4 stars
* Row 3 → Print 3 stars
* Row 2 → Print 2 stars
* Row 1 → Print 1 star

---

## For Loop

```python
for row in range(5, 0, -1):
    for col in range(row):
        print("*", end="")
    print()
```

---

## While Loop

```python
row = 5

while row >= 1:

    col = 0

    while col < row:
        print("*", end="")
        col += 1

    print()
    row -= 1
```

# Pattern 17: Reverse Number Triangle

## Output

```text
12345
1234
123
12
1
```

## Observation

| Row | Output |
| --- | ------ |
| 5   | 12345  |
| 4   | 1234   |
| 3   | 123    |
| 2   | 12     |
| 1   | 1      |

The numbers always start from 1.

The number of digits printed depends on the row number.

---

## Logic

* Row 5 → Print 1 to 5
* Row 4 → Print 1 to 4
* Row 3 → Print 1 to 3
* Row 2 → Print 1 to 2
* Row 1 → Print 1

---

## For Loop

```python
for row in range(5, 0, -1):
    for col in range(1, row + 1):
        print(col, end="")
    print()
```

---

## While Loop

```python
row = 5

while row >= 1:

    col = 1

    while col <= row:
        print(col, end="")
        col += 1

    print()
    row -= 1
```

# Pattern 18: Reverse Alphabet Triangle

## Output

```text
ABCDE
ABCD
ABC
AB
A
```

## Observation

| Row | Output |
| --- | ------ |
| 5   | ABCDE  |
| 4   | ABCD   |
| 3   | ABC    |
| 2   | AB     |
| 1   | A      |

The pattern behaves exactly like Pattern 17.

The only difference is that we print alphabets instead of numbers.

---

## For Loop

```python
for row in range(5, 0, -1):
    for col in range(row):
        print(chr(65 + col), end="")
    print()
```

---

## While Loop

```python
row = 5

while row >= 1:

    col = 0

    while col < row:
        print(chr(65 + col), end="")
        col += 1

    print()
    row -= 1
```

---

## Easier Version

```python
letters = "ABCDE"

for row in range(5, 0, -1):
    for col in range(row):
        print(letters[col], end="")
    print()
```

# General Formula for Reverse Patterns

Almost every reverse triangle pattern follows this structure:

```python
for row in range(n, 0, -1):
    for col in range(row):
        print(..., end="")
    print()
```

# Pattern Analysis Summary

| Pattern                            | Printed Value   | Depends On |
| ---------------------------------- | --------------- | ---------- |
| Reverse Star Triangle              | *               | Row        |
| Reverse Number Triangle            | Column Number   | Column     |
| Reverse Repeated Number Triangle   | Row Number      | Row        |
| Reverse Alphabet Triangle          | Column Position | Column     |
| Reverse Repeated Alphabet Triangle | Row Position    | Row        |

---

# Key Takeaways

To solve any reverse triangle pattern:

### Step 1

Count the rows.

### Step 2

Start rows from N and move towards 1.

### Step 3

Determine what to print:

* Star
* Number
* Alphabet
* Row value
* Column value

### Step 4

Write the outer loop.

### Step 5

Write the inner loop.

Remember:

```text
Reverse Pattern = Decreasing Rows + Columns + Logic
```
