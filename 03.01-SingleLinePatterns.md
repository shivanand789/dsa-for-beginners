# Level 1: Single Line Patterns

Before learning complex patterns, we start with **Single Line Patterns**.

In Level 1, there is only **one row**.

The goal is to understand:

* How loops work
* How repetition works
* The difference between `for` and `while` loops
* How to convert a `for` loop into a `while` loop

---

## Pattern 1: Print 5 Stars

### Output

```text
*****
```

### Observation

* We need to print `*`
* We need to print it **5 times**
* Since the same action repeats, we use a loop

---

### Using a For Loop

```python
for i in range(0, 5, 1):
    print("*", end="")
```

### Using a While Loop

```python
i = 0

while i < 5:
    print("*", end="")
    i += 1
```

---

## Converting a For Loop to a While Loop

### General Formula

### For Loop

```python
for i in range(start, stop, step):
    # code
```

### Equivalent While Loop

```python
i = start

while i < stop:
    # code
    i += step
```

### Understanding the While Loop

Every while loop has three parts:

```python
i = 0          # Initialization

while i < 5:   # Condition
    ...
    i += 1     # Increment
```

### Example

```python
for i in range(1, 6):
    print(i)
```

becomes

```python
i = 1

while i < 6:
    print(i)
    i += 1
```

# Pattern 2: Print Number 1 Five Times

### Output

```text
1 1 1 1 1
```

### Observation

* The value never changes.
* We only repeat the same number five times.

### Using a For Loop

```python
for i in range(5):
    print(1, end=" ")
```

### Using a While Loop

```python
i = 0

while i < 5:
    print(1, end=" ")
    i += 1
```

# Pattern 3: Print Numbers from 1 to 5

### Output

```text
1 2 3 4 5
```

### Observation

* Numbers increase by 1.
* Loop variable itself can be printed.

### Using a For Loop

```python
for i in range(1, 6):
    print(i, end=" ")
```

### Using a While Loop

```python
i = 1

while i <= 5:
    print(i, end=" ")
    i += 1
```

### Output

```text
1 2 3 4 5
```

# Pattern 5: Print Odd Numbers

### Output

```text
1 3 5 7 9
```

---

## Method 1: Using Step Size

### For Loop

```python
for i in range(1, 10, 2):
    print(i, end=" ")
```

### While Loop

```python
i = 1

while i <= 9:
    print(i, end=" ")
    i += 2
```

---

## Method 2: Using Condition Check

### For Loop

```python
for i in range(1, 10):
    if i % 2 != 0:
        print(i, end=" ")
```

### While Loop

```python
i = 1

while i <= 9:
    if i % 2 != 0:
        print(i, end=" ")
    i += 1
```
