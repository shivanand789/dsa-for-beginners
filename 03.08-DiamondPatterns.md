## Pattern 30: Diamond Star Pattern

### Output

```text
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```

This pattern is a combination of:

1. A Normal Star Pyramid
2. An Inverted Star Pyramid

Think of it as:

```text
Star Pyramid
      +
Inverted Star Pyramid
      =
Diamond Pattern
```

---

### Universal Pattern Rule

For every pattern problem:

```text
for each row
    for each column
        decide:
            print symbol?
            or print space?
```

---

### Pattern Analysis

#### Upper Half

| Row | Spaces | Stars |
| --- | ------ | ----- |
| 0   | 4      | 1     |
| 1   | 3      | 3     |
| 2   | 2      | 5     |
| 3   | 1      | 7     |
| 4   | 0      | 9     |

Formula:

```text
Spaces = n - row - 1

Stars = 2 × row + 1
```

---

#### Lower Half

| Row | Spaces | Stars |
| --- | ------ | ----- |
| 5   | 1      | 7     |
| 6   | 2      | 5     |
| 7   | 3      | 3     |
| 8   | 4      | 1     |

Formula:

```text
Spaces = n - row - 1

Stars = 2 × row + 1
```

The lower half simply reverses the upper half.

---

---

### For Loop

```python
n = 5

# Upper Half
for row in range(n):

    spaces = n - row - 1
    stars = 2 * row + 1

    print(" " * spaces + "*" * stars)

# Lower Half
for row in range(n - 2, -1, -1):

    spaces = n - row - 1
    stars = 2 * row + 1

    print(" " * spaces + "*" * stars)
```

---

### While Loop

```python
n = 5

# Upper Half
row = 0

while row < n:

    spaces = n - row - 1
    stars = 2 * row + 1

    print(" " * spaces + "*" * stars)

    row += 1

# Lower Half
row = n - 2

while row >= 0:

    spaces = n - row - 1
    stars = 2 * row + 1

    print(" " * spaces + "*" * stars)

    row -= 1
```

## Pattern 31: Diamond Number Pyramid

### Output

```text
    1
   121
  12321
 1234321
123454321
 1234321
  12321
   121
    1
```

This pattern is a combination of:

1. Number Pyramid
2. Inverted Number Pyramid

---

### Observation

Look at Row 4:

```text
123454321
```

Break it into two parts:

#### Increasing Part

```text
12345
```

Generated using:

```python
for num in range(1, row + 1):
```

---

#### Decreasing Part

```text
4321
```

Generated using:

```python
for num in range(row - 1, 0, -1):
```

---

### Example

Suppose:

```text
row = 4
```

Increasing:

```text
1234
```

Decreasing:

```text
321
```

Combined:

```text
1234321
```

---

### Pattern Analysis

#### Upper Half

| Row | Spaces | Peak Number |
| --- | ------ | ----------- |
| 1   | 4      | 1           |
| 2   | 3      | 2           |
| 3   | 2      | 3           |
| 4   | 1      | 4           |
| 5   | 0      | 5           |

---

#### Lower Half

| Row | Spaces | Peak Number |
| --- | ------ | ----------- |
| 4   | 1      | 4           |
| 3   | 2      | 3           |
| 2   | 3      | 2           |
| 1   | 4      | 1           |

---

### Logic

Every row consists of:

```text
Spaces
+
Increasing Numbers
+
Decreasing Numbers
```

---

### For Loop

```python
n = 5

# Upper Half
for row in range(1, n + 1):

    print(" " * (n - row), end="")

    # Increasing Numbers
    for num in range(1, row + 1):
        print(num, end="")

    # Decreasing Numbers
    for num in range(row - 1, 0, -1):
        print(num, end="")

    print()

# Lower Half
for row in range(n - 1, 0, -1):

    print(" " * (n - row), end="")

    # Increasing Numbers
    for num in range(1, row + 1):
        print(num, end="")

    # Decreasing Numbers
    for num in range(row - 1, 0, -1):
        print(num, end="")

    print()
```

---

### While Loop

```python
n = 5

# Upper Half
row = 1

while row <= n:

    print(" " * (n - row), end="")

    num = 1
    while num <= row:
        print(num, end="")
        num += 1

    num = row - 1
    while num >= 1:
        print(num, end="")
        num -= 1

    print()
    row += 1

# Lower Half
row = n - 1

while row >= 1:

    print(" " * (n - row), end="")

    num = 1
    while num <= row:
        print(num, end="")
        num += 1

    num = row - 1
    while num >= 1:
        print(num, end="")
        num -= 1

    print()
    row -= 1
```

---

### Pattern Formula

A Diamond Number Pattern is:

```text
Spaces
+
Increasing Numbers
+
Decreasing Numbers
```

Repeated twice:

```text
Upper Pyramid
+
Lower Pyramid
```
