# Level 6: Pyramid Patterns

Until now, we worked with:

* Square patterns
* Right triangle patterns
* Reverse triangle patterns

In this level, we will learn **Pyramid Patterns**.

Pyramid patterns introduce a new concept:

* Spaces
* Symmetry

Most pyramid problems can be solved using three parts:

1. Print Spaces
2. Print Left Side
3. Print Right Side

---

## Pattern 25: Star Pyramid

### Output

```text
    *
   ***
  *****
 *******
*********
```

### Observation

| Row | Spaces | Stars |
| --- | ------ | ----- |
| 1   | 4      | 1     |
| 2   | 3      | 3     |
| 3   | 2      | 5     |
| 4   | 1      | 7     |
| 5   | 0      | 9     |

Notice:

* Spaces decrease by 1.
* Stars increase by 2.

### Formula

```text
Spaces = rows - row

Stars = (2 × row) - 1
```

Example:

```text
Row 1 → 4 spaces, 1 star
Row 2 → 3 spaces, 3 stars
Row 3 → 2 spaces, 5 stars
```

---

**For Loop**

```python
rows = 5

for row in range(1, rows + 1):

    # Print spaces
    for space in range(rows - row):
        print(" ", end="")

    # Print stars
    for star in range((2 * row) - 1):
        print("*", end="")

    print()
```

---

**While Loop**

```python
rows = 5
row = 1

while row <= rows:

    spaces = rows - row

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    stars = 1

    while stars <= (2 * row - 1):
        print("*", end="")
        stars += 1

    print()
    row += 1
```

## Pattern 26: Number Pyramid

### Output

```text
    1
   121
  12321
 1234321
123454321
```

### Observation

Look at Row 5:

```text
123454321
```

Notice:

* The middle number is always the row number.
* Numbers increase from 1 to row.
* Numbers then decrease from row - 1 back to 1.

### Pattern Breakdown

For Row 5:

```text
1 2 3 4 5 4 3 2 1
```

Think of it as:

```text
Increasing Part
1 → row

Decreasing Part
row - 1 → 1
```

---

### Logic

#### Spaces

Same as the previous pyramid:

```python
for space in range(rows - row):
```

#### Increasing Numbers

```python
for num in range(1, row + 1):
```

#### Decreasing Numbers

```python
for num in range(row - 1, 0, -1):
```

---

**For Loop**

```python
rows = 5

for row in range(1, rows + 1):

    # Spaces
    for space in range(rows - row):
        print(" ", end="")

    # Increasing numbers
    for num in range(1, row + 1):
        print(num, end="")

    # Decreasing numbers
    for num in range(row - 1, 0, -1):
        print(num, end="")

    print()
```

---

**While Loop**

```python
rows = 5
row = 1

while row <= rows:

    spaces = rows - row

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    num = 1

    while num <= row:
        print(num, end="")
        num += 1

    num = row - 1

    while num >= 1:
        print(num, end="")
        num -= 1

    print()
    row += 1
```

## Pattern 27: Alphabet Pyramid

### Output

```text
    A
   ABA
  ABCBA
 ABCDCBA
ABCDEDCBA
```

### Observation

This pattern follows exactly the same logic as the Number Pyramid.

The only difference:

* Numbers are replaced with letters.

### Row Analysis

Row 5:

```text
ABCDEDCBA
```

Break it into two parts:

```text
ABCDE
```

and

```text
DCBA
```

### Logic

#### Spaces

```python
rows - row
```

#### Increasing Letters

```python
A → row letter
```

#### Decreasing Letters

```python
row - 1 letter → A
```

---

**For Loop**

```python
rows = 5

for row in range(1, rows + 1):

    # Spaces
    for space in range(rows - row):
        print(" ", end="")

    # Increasing letters
    for col in range(1, row + 1):
        print(chr(64 + col), end="")

    # Decreasing letters
    for col in range(row - 1, 0, -1):
        print(chr(64 + col), end="")

    print()
```

---

**While Loop**

```python
rows = 5
row = 1

while row <= rows:

    spaces = rows - row

    while spaces > 0:
        print(" ", end="")
        spaces -= 1

    col = 1

    while col <= row:
        print(chr(64 + col), end="")
        col += 1

    col = row - 1

    while col >= 1:
        print(chr(64 + col), end="")
        col -= 1

    print()
    row += 1
```

## Key Takeaways

Before solving any pyramid pattern:

### Step 1

Count the number of spaces.

```text
spaces = rows - row
```

### Step 2

Identify the middle element.

Examples:

```text
*
5
E
```

### Step 3

Determine:

* What increases?
* What decreases?

### Step 4

Write the pattern in three parts:

```text
Spaces
+
Left Side
+
Right Side
```

Remember:

```text
Pyramid Pattern = Spaces + Symmetry + Logic
```

Once you understand this formula, most pyramid patterns become much easier to solve.
