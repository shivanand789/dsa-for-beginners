## Pattern 37: Butterfly Pattern

### Output

```text
*      *
**    **
***  ***
********
********
***  ***
**    **
*      *
```

---

### Observation

The pattern has two parts:

#### Upper Half

```text
*      *
**    **
***  ***
********
```

#### Lower Half

```text
********
***  ***
**    **
*      *
```

Notice:

* Stars increase on both sides.
* Spaces decrease in the middle.
* Lower half is the reverse of the upper half.

---

### Pattern Analysis

For n = 4

| Row | Left Stars | Spaces | Right Stars |
| --- | ---------- | ------ | ----------- |
| 1   | 1          | 6      | 1           |
| 2   | 2          | 4      | 2           |
| 3   | 3          | 2      | 3           |
| 4   | 4          | 0      | 4           |

Formula:

```text
Left Stars = row

Spaces = 2 × (n - row)

Right Stars = row
```

---

### For Loop

```python
n = 4

# Upper Half
for row in range(1, n + 1):

    print("*" * row, end="")
    print(" " * (2 * (n - row)), end="")
    print("*" * row)

# Lower Half
for row in range(n, 0, -1):

    print("*" * row, end="")
    print(" " * (2 * (n - row)), end="")
    print("*" * row)
```

---

### While Loop

```python
n = 4

# Upper Half
row = 1

while row <= n:

    print("*" * row, end="")
    print(" " * (2 * (n - row)), end="")
    print("*" * row)

    row += 1

# Lower Half
row = n

while row >= 1:

    print("*" * row, end="")
    print(" " * (2 * (n - row)), end="")
    print("*" * row)

    row -= 1
```

---

### Key Idea

```text
Butterfly Pattern

=
Upper Triangle
+
Lower Triangle
```

## Pattern 38: X Pattern

### Output

```text
*   *
 * *
  *
 * *
*   *
```

---

### Observation

Stars appear only on the diagonals.

For a 5 × 5 grid:

| Condition          | Meaning            |
| ------------------ | ------------------ |
| row == col         | Main diagonal      |
| row + col == n - 1 | Secondary diagonal |

Print a star when either condition is true.

Otherwise print a space.

---

### Visual Analysis

```text
*   *   ← row 0
 * *    ← row 1
  *     ← row 2
 * *    ← row 3
*   *   ← row 4
```

---

### For Loop

```python
n = 5

for row in range(n):

    for col in range(n):

        if row == col or row + col == n - 1:
            print("*", end="")
        else:
            print(" ", end="")

    print()
```

---

### While Loop

```python
n = 5

row = 0

while row < n:

    col = 0

    while col < n:

        if row == col or row + col == n - 1:
            print("*", end="")
        else:
            print(" ", end="")

        col += 1

    print()
    row += 1
```

---

### Key Idea

```text
X Pattern

=
Main Diagonal
+
Secondary Diagonal
```

## Pattern 39: Hourglass Pattern

### Output

```text
*****
 ***
  *
 ***
*****
```

---

### Observation

The pattern has:

* An upper inverted pyramid
* A lower pyramid

---

### Pattern Analysis

Upper Half:

```text
*****
 ***
  *
```

Lower Half:

```text
 ***
*****
```

---

### For Loop

```python
n = 3

# Upper Half
for row in range(n):

    print(" " * row, end="")
    print("*" * (2 * (n - row) - 1))

# Lower Half
for row in range(1, n):

    print(" " * (n - row - 1), end="")
    print("*" * (2 * row + 1))
```

---

### While Loop

```python
n = 3

# Upper Half
row = 0

while row < n:

    print(" " * row, end="")
    print("*" * (2 * (n - row) - 1))

    row += 1

# Lower Half
row = 1

while row < n:

    print(" " * (n - row - 1), end="")
    print("*" * (2 * row + 1))

    row += 1
```

---

### Key Idea

```text
Hourglass

=
Inverted Pyramid
+
Normal Pyramid
```

## Pattern 40: Zig-Zag Pattern

### Output

```text
*   *   *
 * * * *
  *   *
```

---

### Observation

This pattern is commonly asked in coding interviews.

It is usually printed using:

* 3 rows
* N columns

Stars appear at specific positions.

For 3 rows:

```text
Row 0 → Peaks

Row 1 → Diagonal Connections

Row 2 → Valleys
```

---

### Visual Analysis

```text
*   *   *

 * * * *

  *   *
```

Notice:

* Top row contains peak stars.
* Middle row connects the peaks.
* Bottom row contains valley stars.

---

### For Loop

```python
n = 9

for row in range(3):

    for col in range(n):

        if (
            (row + col) % 4 == 0
            or
            (row == 1 and col % 4 == 2)
        ):
            print("*", end="")
        else:
            print(" ", end="")

    print()
```

---

### While Loop

```python
n = 9

row = 0

while row < 3:

    col = 0

    while col < n:

        if (
            (row + col) % 4 == 0
            or
            (row == 1 and col % 4 == 2)
        ):
            print("*", end="")
        else:
            print(" ", end="")

        col += 1

    print()
    row += 1
```

## Key Takeaways

Before solving advanced patterns, ask:

### Step 1

Can this pattern be split into smaller patterns?

Examples:

```text
Diamond
=
Pyramid + Inverted Pyramid

Butterfly
=
Upper Triangle + Lower Triangle

Hourglass
=
Inverted Pyramid + Pyramid
```

### Step 2

Look for symmetry.

### Step 3

Identify:

* Spaces
* Stars
* Diagonals
* Boundaries
