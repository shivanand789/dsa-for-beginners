# LeetCode 344 - Reverse String

## Problem

Write a function that reverses a string.

The input string is given as an array of characters `s`.

You must modify the input array **in-place** with **O(1)** extra memory.

### Example

```python
s = ["h","e","l","l","o"]
```

Output:

```python
["o","l","l","e","h"]
```

---

# Optimal Solution (Two Pointers)

```python
from typing import List

class Solution:
    def reverseString(self, s: List[str]) -> None:
        l = 0
        r = len(s) - 1

        while l < r:
            s[l], s[r] = s[r], s[l]

            l += 1
            r -= 1
```

---

# Intuition

Use two pointers:

```text
h e l l o
↑       ↑
L       R
```

Swap first and last.

```text
o e l l h
  ↑   ↑
  L   R
```

Swap again.

```text
o l l e h
    ↑
```

Done.

# Dry Run

Input:

```python
s = ["h","e","l","l","o"]
```

Initial:

```text
l = 0
r = 4
```

---

## Iteration 1

Swap:

```python
s[0], s[4] = s[4], s[0]
```

Result:

```python
["o","e","l","l","h"]
```

Move pointers:

```python
l = 1
r = 3
```

---

## Iteration 2

Swap:

```python
s[1], s[3] = s[3], s[1]
```

Result:

```python
["o","l","l","e","h"]
```

Move pointers:

```python
l = 2
r = 2
```

Stop because:

```python
l < r
```

is no longer true.

---

# Why Does This Work?

Every swap places:

* One character in its correct position from the front
* One character in its correct position from the back

After `n/2` swaps, the entire array is reversed.

---

# Time Complexity

```text
O(n)
```

Each character is visited at most once.

---

# Space Complexity

```text
O(1)
```

No extra array is used.

The reversal happens in-place.

---

# Interview Explanation

"I use two pointers, one starting at the beginning and one at the end of the array. I repeatedly swap the characters at those positions and move the pointers toward the center. Once the pointers meet, the string is completely reversed. This solution runs in O(n) time and O(1) space."
